# Code of Conduct

This project expects respectful, constructive, and professional collaboration.

## Expected behavior
- Be respectful and constructive in issues and pull requests.
- Focus feedback on code and behavior, not people.
- Assume good intent and ask clarifying questions.

## Unacceptable behavior
- Harassment, hate speech, or personal attacks.
- Trolling, intimidation, or doxxing.
- Sharing private data or undisclosed vulnerabilities publicly.

## Enforcement
Maintainers may remove content, close threads, or block contributors for repeated violations.
